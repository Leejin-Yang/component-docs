# Container

화면의 폭에 따라 내용의 폭을 조절하고 중앙에 배치하는 레이아웃 컴포넌트입니다.

```jsx static
import { Container } from 'layout-component-hp';
```

## Usage

`Container` 컴포넌트의 사용법은 다음과 같습니다.

```jsx live
<Container
  minWidth={100}
  maxWidth={600}
  css={{ padding: '10px', backgroundColor: '#f0f0f0' }}
>
  <div>This is a container.</div>
</Container>
```

### Centering the children

centerContent 속성을 사용하여 자식 요소를 수평 및 수직으로 중앙에 배치할 수 있습니다.

```jsx live
<Container
  minWidth={100}
  maxWidth={600}
  css={{ padding: '10px', backgroundColor: '#f0f0f0' }}
  centerContent
>
  <div>This is a container.</div>
</Container>
```
